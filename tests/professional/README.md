## 🎯 **QUICK ACCESS**

### 🚀 **Hlavní testování**
```bash
# Otevřít profesionální test suite
open tests/COMPLETE_WEBSITE_TEST_SUITE.html

# Spustit web pro testování
npm run dev
open http://localhost:5173
```

```bash
# Specifické tooltip testy

```
